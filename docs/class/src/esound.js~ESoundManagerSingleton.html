    public createSound(pSoundPath: number, pVolume: number, pStartTime: number, pEndTime: number, pSave: boolean, pPlayUnfocused: boolean, pPlaybackRate: number, pLoop: boolean, pCallbackObject: Object): Sound source

    Params:

    NameTypeAttributeDescription
    pSoundPath number

    The path of the sound file
    pVolume number
    • optional
    • default: 100

    The volume of the sound
    pStartTime number
    • optional
    • default: 0

    The start time of this sound (to play a clipped version)
    pEndTime number
    • optional
    • default: duration

    The end time of this sound (to play a clipped version)
    pSave boolean
    • optional
    • default: false

    Whether to save this sound, or recycle it when it's completed
    pPlayUnfocused boolean
    • optional
    • default: false

    If this sound is set to true then it will not be paused automatically when the game screen is not focused
    pPlaybackRate number
    • optional
    • default: 1

    The rate at which the sound is played, Higher numbers for faster playback (MAX 10)
    pLoop boolean
    • optional
    • default: false

    Whether this sound should loop or not
    pCallbackObject Object

    Object full of callbacks to call (soon to be deprecated and replaced with on('event') handlers)
    pCallbackObject.onStarted function

    Callback for when the sound has started playing
    pCallbackObject.onStopped function

    Callback for when the sound has stopped playing
    pCallbackObject.onEnded function

    Callback for when the sound has ended
    pCallbackObject.onSuspended function

    Callback for when the sound has been paused
    pCallbackObject.onResumed function

    Callback for when the sound has been resumed

    Return:

    Sound

    A sound object that has vast API on controlling it

    public emit(pSoundPath: string, pVolume: number, pStartTime: number, pEndTime: number, pPlaybackRate: number): SoundSource source

    Plays a sound that is not stored. Plays the sound only. You can only kill this sound, no repeat, no changing volume, etc

    Params:

    NameTypeAttributeDescription
    pSoundPath string

    The path to the file
    pVolume number
    • optional
    • default: 100

    The volume to play the sound
    pStartTime number
    • optional
    • default: 0

    The start time of this sound (to play a clipped version)
    pEndTime number
    • optional
    • default: duration

    The end time of this sound (to play a clipped version)
    pPlaybackRate number
    • optional
    • default: 1

    The rate at which the sound is played, Higher numbers for faster playback (MAX 10)

    Return:

    SoundSource

    The source to this emitted sound. Call source.kill() to stop this sound while its playing. This is the only API this sound has
